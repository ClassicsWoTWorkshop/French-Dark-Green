Thank you for downloading!

To install:
- Drag the "gui", "scripts", and "vehicles" folders from inside the included "MAIN MOD (Install this FIRST)" folder to your World of Tanks install directory (by default, C:\Games\World_of_Tanks\res_mods\<version>) (example: C:\Games\World_Of_Tanks\res_mods\0.9.20.0).

Optional:
- If you want to remove the AMX M4 49 Liberte's skin, drag the "gui" and "vehicles" folders from inside the "ADDON - Remove Liberte Skin" folder to <Your WoT Directory\res_mods\<Game Version>\ (example: C:\Games\World_of_Tanks\res_mods\0.9.20.0\).

Enjoy!
